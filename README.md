# Purchase Order Full Stack App

## Tech Stack

* Backend: ASP.NET Core Web API
* Frontend: Angular
* Database: Entity Framework Core

## Features

* Manage Locations
* API integration with Angular
* CRUD operations

## How to Run

### Backend

cd Backend/BackendAPI
dotnet run

### Frontend

cd frontend
npm install
ng serve

## API Endpoint

https://localhost:5001/api/location
